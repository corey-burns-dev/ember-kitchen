# Ember Kitchen

Ember Kitchen is a restaurant website built with Astro. It combines a refined dining brand, menu presentation, reservation flow, and private dining pages in a compact marketing site.

## Highlights

- Home, menu, reservations, about, and private dining pages
- Strong editorial art direction with restaurant-focused content blocks
- Mostly static Astro pages with light interactive behavior

## Stack

- Astro
- React
- Tailwind CSS

## Quick start

```bash
bun install
bun run dev
```

Checks and build:

```bash
bun run typecheck
bun run build
```
